What is this?

This website is meant to be a proof-of-concept that an accurate and easy to use couse designing platform can be built using modern web technologies. It is a stepping stone to a larger system that will build out a more fully featured (and better looking) web app.

How do I use it?

To start, click the Settings button, and enter the dimensions of your ring, rail length, and width for spread elements (oxers only at this point).
Once the ring is set up, there are a few basic operations:

  • Double click to add a new jump ( currently only verticals, but more to come! )
  • Click on a jump to select it. Once a jump is selected, it can be rotated by scrolling
  • Click and drag to move a jump around
A list of jumps can be found at the bottom of the page, and from there the precise position, angle, spread, and rail length of each jump can be found. These values can be changed for each jump, and the course map is refreshed when update is clicked. The jump can also be deleted from here. The x and y values given locate the center of each jump, relative to the top left corner.
There are also several tools shown below the ring. They include:

  • Measure: measures the straight line distance in feet between the center of two jumps. Click on the measure button, if no first jump is seleced, click on a starting jump, then click on the second jump to see the straight line distance between them. Clicking on more jumps will chain them together.
  • Path Measure: estimates the path between two jumps and calculates the distance for that path in feet. Click on the path measure button, if no first jump is seleced, click on a starting jump, then click on the second jump to see the straight line distance between them.
  • Jump Type: select they type of jump that will be placed on next double click. Currently only supports oxers and verticals, but more are coming!
